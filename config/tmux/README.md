# tmux 設定

## ファイル構成

```
config/tmux/
└── tmux.conf    # メイン設定ファイル
```

## 基本設定

| 設定 | 値 |
|------|----|
| Prefix キー | `C-a`（デフォルトの `C-b` から変更） |
| ターミナル | `tmux-256color` + True Color |
| インデックス開始 | 1（ウィンドウ・ペイン共通） |
| スクロールバック | 100,000 行 |
| マウス操作 | 有効 |
| ウィンドウ名 | カレントディレクトリ名に自動設定 |

## キーバインド

### セッション・ウィンドウ操作

| キー | 動作 |
|------|------|
| `Prefix c` | 新規ウィンドウ（カレントディレクトリを引き継ぎ） |
| `Prefix r` | 設定ファイルをリロード |

### ペイン操作

| キー | 動作 |
|------|------|
| `Prefix \|` | 垂直分割（カレントディレクトリを引き継ぎ） |
| `Prefix -` | 水平分割（カレントディレクトリを引き継ぎ） |
| `Prefix x` | ペインを閉じる（確認なし） |
| `Prefix h/j/k/l` | Vim 方向でペイン移動 |
| `Prefix H/J/K/L` | Vim 方向でペインリサイズ（連打可能） |

### コピーモード

| キー | 動作 |
|------|------|
| `Prefix [` | コピーモード開始 |
| `v` | 選択開始（Vim 風） |
| `y` | コピーしてモード終了 |

## 外観

- **ステータスバー左**: セッション名（緑）
- **ステータスバー右**: Git ブランチ名（青）/ 日付（黄）/ 時刻（シアン）
- **アクティブペインボーダー**: `#89b4fa`（ブルー）
- **非アクティブペインボーダー**: `#444444`（グレー）
