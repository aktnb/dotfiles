# キーバインド一覧

## tmux

プリフィックスキー: `Ctrl-a`

### 基本操作

| キー | 機能 |
|------|------|
| `prefix + r` | 設定ファイル再読み込み |
| `prefix + x` | ペインを閉じる（確認なし） |

### ペイン操作

| キー | 機能 |
|------|------|
| `prefix + \|` | ペインを水平分割（カレントディレクトリ引き継ぎ） |
| `prefix + -` | ペインを垂直分割（カレントディレクトリ引き継ぎ） |
| `prefix + h` | 左のペインに移動 |
| `prefix + j` | 下のペインに移動 |
| `prefix + k` | 上のペインに移動 |
| `prefix + l` | 右のペインに移動 |
| `prefix + H` | ペインを左にリサイズ（連打可） |
| `prefix + J` | ペインを下にリサイズ（連打可） |
| `prefix + K` | ペインを上にリサイズ（連打可） |
| `prefix + L` | ペインを右にリサイズ（連打可） |

### ウィンドウ操作

| キー | 機能 |
|------|------|
| `prefix + c` | 新規ウィンドウ（カレントディレクトリ引き継ぎ） |

### コピーモード

| キー | 機能 |
|------|------|
| `prefix + v` | 選択開始 |
| `prefix + y` | 選択をコピーしてコピーモード終了 |

## Neovim

リーダーキー: `Space`

### 基本操作

| キー | モード | 機能 |
|------|--------|------|
| `<leader>w` | Normal | ファイルを保存 |
| `<leader>q` | Normal | ファイルを閉じる |
| `<leader>h` | Normal / Visual | 行頭に移動 |
| `<leader>l` | Normal / Visual | 行末に移動 |
| `<Esc><Esc>` | Normal | 検索ハイライトをクリア |
| `jj` | Insert | Insertモード終了 |
| `p` | Visual | ペースト（レジスタを上書きしない） |

### LSP

| キー | モード | 機能 |
|------|--------|------|
| `gd` | Normal | 定義にジャンプ |
| `gD` | Normal | 宣言にジャンプ |
| `gi` | Normal | 実装にジャンプ |
| `gr` | Normal | 参照一覧 |
| `K` | Normal | ホバー情報を表示 |
| `<C-k>` | Normal | シグネチャヘルプを表示 |
| `<leader>rn` | Normal | リネーム |
| `<leader>ca` | Normal | コードアクション |
| `<leader>f` | Normal | フォーマット |
| `[d` | Normal | 前の診断へ移動 |
| `]d` | Normal | 次の診断へ移動 |
| `<leader>d` | Normal | 診断情報をフロートウィンドウで表示 |

### 補完（nvim-cmp）

| キー | モード | 機能 |
|------|--------|------|
| `<C-Space>` | Insert | 補完メニューを開く |
| `<C-e>` | Insert | 補完メニューを閉じる |
| `<CR>` | Insert | 補完候補を確定 |
| `<Tab>` | Insert / Select | 次の候補を選択 / スニペット展開・次へ |
| `<S-Tab>` | Insert / Select | 前の候補を選択 / スニペット前へ |
| `<C-b>` | Insert | ドキュメントを上にスクロール |
| `<C-f>` | Insert | ドキュメントを下にスクロール |

### コメント（Comment.nvim）

| キー | モード | 機能 |
|------|--------|------|
| `gcc` | Normal | 行コメントをトグル |
| `gbc` | Normal | ブロックコメントをトグル |
| `gc` | Visual / Operator | 選択範囲の行コメントをトグル |
| `gb` | Visual / Operator | 選択範囲のブロックコメントをトグル |

### ファイルエクスプローラ（neo-tree）

| キー | モード | 機能 |
|------|--------|------|
| `<leader>e` | Normal | ファイルエクスプローラーを表示/非表示 |
| `<leader>o` | Normal | ファイルエクスプローラーにフォーカス |

## Zsh

### Zeno

| キー | 機能 |
|------|------|
| `Space` | スニペット自動展開 |
| `<C-m>` | スニペット展開 + 実行 |
| `<C-x><x>` | スニペット挿入 |
| `<C-x>Space` | スペース挿入 |
| `<C-x><C-m>` | コマンドライン実行 |
| `<C-x><C-z>` | 自動スニペットのトグル |
| `<C-x>p` | Preprompt 実行 |
| `<C-x>s` | Preprompt スニペット実行 |
| `<C-r>` | ヒストリー検索（fzf） |
| `<C-g>` | ghq リポジトリ移動 |
